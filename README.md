#  a blog for vue  

##      技术 vue2.0 vuex sass es6 vue-router2.0
###  安装
####         git clone https://github.com/xyxiao001/vue-blog.git
####         cd vue--blog
####         npm install -d
###  启动
####         npm run dev
###  编译
####         npm run build
